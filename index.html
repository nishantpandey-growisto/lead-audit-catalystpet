  • N/A -- US market, no COD expected
  • Afterpay available but not prominently displayed. Consider adding BNPL messaging on PDP and cart ('4 payments of $X with Afterpay') to reduce price sensitivity for larger subscription orders.
Image Optimization Needed

CDN & Performance

Shopify CDN (cdn.shopify.com)

  • Images served via Shopify CDN with automatic WebP conversion. However, several product images are large (900x900 originals) and some decorative images lack width/height attributes causing layout shifts.
  • Shopify CDN handles compression automatically. The corrupted Futura-Normal.ttf font file adds unnecessary network requests with repeated download failures.
  • Standard Shopify CDN caching in place. Third-party scripts (Klaviyo, Yotpo, Clarity, Triple Whale, TikTok, Facebook) add significant external requests.
Standard Security

Security & Compliance

Active

  • Full HTTPS enabled across all pages via Shopify's automatic SSL certificate
  • PCI DSS compliant via Shopify Payments. Checkout runs on Shopify's secure infrastructure.
  • Standard Shopify cookie implementation. Multiple tracking scripts (GA4, FB Pixel, TikTok, Klaviyo, Clarity, Triple Whale) set cookies -- privacy policy page exists but no cookie consent banner was observed.

Technology Narrative

Catalyst Pet runs on Shopify with a custom-built theme ('Catalyst' v1.2.11) that shows signs of technical debt. The theme relies on jQuery 1.10.0 and other legacy libraries from 2013, which impacts page performance. A corrupted font file (Futura-Normal.ttf) generates 8+ console warnings per page load, and a missing JavaScript file (bio-bag-validation.js) causes additional errors. The checkout infrastructure is solid with Shopify Payments, express checkout options, and Afterpay BNPL -- but these payment options are not prominently surfaced in the shopping experience. The most critical opportunity is modernizing the theme to Shopify 2.0 architecture, which would unlock section-based customization, improve Core Web Vitals, and eliminate the legacy library dependencies.

App Stack Assessment

Catalyst Pet's app ecosystem is well-established for a subscription-first D2C brand. The core stack of Yotpo (reviews) + Klaviyo (email) + ReCharge (subscriptions) + Gorgias (support) covers the essential customer lifecycle touchpoints. Multi-channel advertising is well-instrumented with Facebook Pixel, TikTok Pixel, Pinterest Tag, and Triple Whale for attribution. The critical issue is triple-redundancy in heatmap/session recording tools: Microsoft Clarity (free), Hotjar, and Lucky Orange all serve the same purpose. Removing two of these would save $30-60/month and reduce JavaScript bloat by ~200KB. Key missing capabilities include a loyalty/rewards program (important for reducing subscription churn), post-purchase upsells (to increase AOV on complementary products like waste bags), and A/B testing (essential for optimizing the subscription vs one-time messaging). Overall, the app infrastructure is solid but could be streamlined for better performance.
